For Real Estate Agents

The AI Vocab Cheatsheet.

Plain English. No jargon. The terms your team is using — so you can stop nodding through every AI conversation.

Foundations.
LLM
The "brain" behind tools like ChatGPT, Claude, and Gemini.
Ask Claude to write a listing description from your MLS data.
Prompt
What you type to the AI. Better prompts get better answers.
"Write a 150-word listing description for a 4-bed in Woodbury, MN."
Token
How LLMs count text. Roughly ¾ of a word.
A full purchase agreement ≈ 8,000 tokens. Plan your prompts around it.
Context Window
The AI's working memory. Bigger = longer docs without forgetting.
Drop a 40-page inspection report in. Ask: "Flag every red flag."
Hallucination
When AI confidently invents facts. The #1 risk to verify against.
AI made up the school rating. Never send AI numbers to a client unchecked.
Mechanics.
Agent
AI that takes action, not just answers. Books, sends, pulls.
An agent watches your sphere, drafts replies, queues them in your CRM.
MCP
The "USB port" connecting AI to Gmail, Drive, Follow Up Boss.
"Claude, summarize every conversation with the Henderson lead."
API
The plumbing that lets software tools talk to each other.
Zapier uses APIs to push new Zillow leads straight into Follow Up Boss.
Embedding
Turning words into numbers so AI can compare meaning.
"Find me a past listing that feels like 4521 Oak Ave." — that's embeddings.
Visibility.
SEO
The old game. Writing so Google ranks you for keywords.
Your blog "Best Schools in Woodbury" ranks #1 and pulls leads monthly.
GEO
The new game. Writing so AI engines cite you in their answers.
A buyer asks ChatGPT "best agent in Woodbury?" Your name shows up.
Backlinks
Other sites linking to yours. Google treats them like votes.
A link from your local chamber of commerce beats 100 spam directories.
AI Crawlers
Bots that read the web for AI. Block them and you disappear.
Allow GPTBot + ClaudeBot in robots.txt so AI quotes your content.
